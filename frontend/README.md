# 🐾 Resgatinhos Unifor

> Plataforma web dedicada a promover a adoção responsável dos gatinhos resgatados no campus da Unifor, em parceria com o curso de Medicina Veterinária.

## 📖 Sobre o Projeto
O **Resgatinhos Unifor** nasceu da necessidade de conectar os felinos resgatados no campus universitário a lares amorosos e seguros. O sistema moderniza o processo de adoção, oferecendo um catálogo dinâmico de animais disponíveis, informações de saúde transparentes e uma futura área administrativa.

## ✨ Funcionalidades

### 🖥️ Frontend (Interface do Usuário)
* **Catálogo Interativo:** Grid responsivo com todos os gatinhos.
* **Filtros Avançados:** Pesquisa em tempo real por nome, sexo e status.
* **Perfis Detalhados:** Páginas individuais com histórico médico completo.
* **UI/UX Moderna:** Animações de scroll, hovers interativos e transições suaves.

### ⚙️ Backend (Em Desenvolvimento)
* API RESTful com Node.js e Express.
* Banco de dados PostgreSQL para gestão de animais e formulários.

## 🚀 Como Executar Localmente

1. **Clone o repositório:**
```bash
git clone https://github.com/lpeixotomagalhaes/adocao-gatos-unifor.git
```
2. **Acesse a pasta:**
```bash
cd adocao-gatos-unifor
```
3. **Instale as dependências:**
```bash
npm install
```
4. **Inicie o projeto:**
```bash
npm run dev
```
